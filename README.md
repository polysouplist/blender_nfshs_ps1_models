# Need for Speed High Stakes (1999) PS1 (.geo) importer/exporter
[![Blender](https://img.shields.io/badge/Blender-v3.6_LTS-blue?logo=blender&logoColor=white)](https://www.blender.org/download/lts/3-6/#versions "Download Blender")

A Blender Add-on that enables importing and exporting vehicles as a Need for Speed High Stakes (1999) PS1 (.geo) file.

## Requirements
- [Blender 3.6 LTS](https://www.blender.org/download/lts/3-6/)
- [Wahooney's Collection Custom Properties Panel](https://www.dropbox.com/s/16obllozlbueqhk/wahooney_custom_properties_collection.py)

## Installation
Go to `Edit > Preferences > Addons > Install` and either select .zip file or the unzipped `import_nfshs_ps1_models.py`, `create_flag_data_panel.py` and `export_nfshs_ps1_models.py` files.

## Location
`File > Import > Need for Speed High Stakes (1999) PS1 (.geo)`
|
`File > Export > Need for Speed High Stakes (1999) PS1 (.geo)`
